# Netflix Titles Dashboard

## Overview
This dashboard analyzes Netflix titles dataset and provides insights on:
- Total Titles
- Movies vs TV Shows
- Titles by Rating
- Top 10 Countries by Titles
- Titles Trend by Release Year

## Tools Used
- Power BI
- CSV Dataset
